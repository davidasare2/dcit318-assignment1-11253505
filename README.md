## Grade Calculator

**Description:**  
This console application prompts the user to enter a numerical grade between 0 and 100. Based on the input, it displays the corresponding letter grade using the following scale:

- 90 and above: A  
- 80–89: B  
- 70–79: C  
- 60–69: D  
- Below 60: F

**How to Run:**  
```bash
cd GradeCalculators
dotnet run
